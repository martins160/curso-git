### Assim que instala o Git use os seguintes comandos

git config --global user.name "martins16"
git config --global user.email "martins160@gmail.com"


### Iniciando o projeto novo

git init

### Adicionando o projeto

git add .

### Comitando os projetos

git commit -m "Início do projeto"

### Enviado alterações para o repositório

git push

### Recebendo alterações do repositório

git pull

### Clonando o repositório

git clone https://github.com/martins160/curso-git.git .

### Log do git

git log

### voltando o status inicial do arquivo antes do PUSH

git checkout Comando.txt
